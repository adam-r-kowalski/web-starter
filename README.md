# Web Starter

## Install with yarn

```
git clone https://github.com/adam-r-kowalski/web-starter.git
cd web-starter
yarn install
yarn dev
open localhost:1234
```

## Install with npm

```
git clone https://github.com/adam-r-kowalski/web-starter.git
cd web-starter
npm install
npm run dev
open localhost:1234
```
